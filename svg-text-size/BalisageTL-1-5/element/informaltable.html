<informaltable> Informal Table

A table to appear without caption or label.
Attributes
border
cellpadding
cellspacing
frame
rules
summary
Models and Context
May be contained in
<appendix>, <article>, <blockquote>, <figure>, <listitem>, <note>, <para>, <section>, <textobject>
Description
The following, in order:
  • Any one of:
    • <col> Column, zero or more
    • <colgroup> Column Group, zero or more
  • <thead> Table Header, zero or one
  • <tfoot> Table Foot, zero or one
  • <tr> Table Row, one or more
Expanded Content Model

((col* | colgroup*), thead?, tfoot?, tr+)
